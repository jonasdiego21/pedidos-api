# pedidos-api
Example of a simple purchase order api

## Technologies Used
 - Javascript
 - Node.js
 - MongoDB
  
## Dependencies
 - bcryptjs 2.4.3
 - cors 2.8.5
 - express 4.17.1
 - jsonwebtoken 8.5.1
 - mongoose 5.6.9
 - mongoose-paginate 5.0.3
 - morgan 1.9.1
 - multer 1.4.2
 - require-dir 1.2.0
  
## How to get the project
```sh
git clone https://github.com/jonasdiego21/pedidos-api.git
```

## How to install the dependencies
```sh
cd pedidos-api
npm install 
```

## How to start the project
```sh
npm dev
npm start
```
